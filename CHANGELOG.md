# Changelog

## [1.0.0] — 2026-06-13

### Added
- **DASSE** (Dynamic AI Smart Stop Engine) — core innovation module
- **Delay Predictor** — LSTM + TFT ensemble (statistical fallback included)
- **RL Controller** — Multi-Agent PPO railway controller with reward function
- **Digital Twin** — National Railway Digital Twin (RailTwin) with simulation
- **IoT Sensor Manager** — 14 sensor type simulators + anomaly detection
- **FastAPI server** — REST API exposing all RailMind AI capabilities
- **Full test suite** — 40+ unit and integration tests
- **CI/CD Pipeline** — GitHub Actions for Python 3.10/3.11/3.12
- Real-time data integration: RailYatri H1 2025, PAC Report Feb 2026, IBEF 2026

### Research Foundation
- Multi-task DRL for Railway Rescheduling (ScienceDirect 2024)
- Causal RL for Single-Track Networks (Elsevier 2025)
- PPO for Train Trajectory Reconstruction (Int. J. Rail Transportation 2024)
